# SpaceChamps

Premium corporate website for SpaceChamps, a drone surveying and geospatial intelligence company operating across Africa and Asia.

## Features

- Responsive Next.js website
- Cinematic drone and LiDAR hero experience
- Africa and Asia coverage visualization
- Services, technology, industries, workflow, and project sections
- Interactive technology showcase
- Project enquiry form experience
- SEO metadata and mobile optimization

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production

```bash
npm run build
npm start
```

The project uses standard Next.js and is ready for deployment on Vercel.
